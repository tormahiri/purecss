جەدۋەل

ئادەتتىكى جەدۋەل (Default Form)

pure-form تۈر نامى (classname) نى ھەرقانداق
ئېلېمېنىتىغا قوشساقلا تورغا كىرىش ئۈچۈن (ئاچقۇچلۇق رولىنى ئوينايدىغان) ئادەتتىكى جەدۋەل (default form) نى ياسىغىلى بولىدۇ. بىر دانە ئىخچام بولغان تور ئاچقۇچ جەدۋىلى (A compact inline form)

ئۈستى-ئۈستىگە (ئارقىمۇ-ئارقا) كەلگەن جەدۋەل (Stacked Form)

ئارقىمۇ-ئارقا كەلگەن جەدۋەل (stacked form)، label ئېلېمىېنتىنىڭ ئاستىدىكى input ئېلېمېنىتلىرى بىلەن ، pure-form-stacked تۈر نامى (classname) نى بىر دانە ئېلېمېنىتىدىكى pure-form نىڭ ئارقىسىغا قوشۇش ئارقىلىق ياسىلىدۇ. بىر دانە ئارقىمۇ-ئارقا كەلگەن جەدۋەل (A Stacked Form)


قەۋەتلىك جەدۋەل (Aligned Form)

قەۋەتلىك جەدۋەل (aligned form) بولسا، pure-form-aligned تۈر نامى (classname) نى ئېلېمېنتىدىكى pure-form نىڭ ئارقىسىغا قوشۇپ ياسىلىدۇ. قەۋەتلىك جەدۋەلدە، label ئېلېمېنتى ئوڭچىل قەۋەت input ئېلېمېنت كونتۇرۇللىقىغا قارىمۇ-قارشى بولۇپ، بىراق كىچىكرەك ئېكرانلاردا ئارقىمۇ-ئارقا كەلگەن جەدۋەل (A Stacked Form) ھالىتىگە قايتىدۇ.


كۆپ ئىستونلۇق جەدۋەل (Multi-Column Form) (Pure Grids لاردىن تۈزۈلگەن)

كۆپ ئىستونلۇق جەدۋەل (multi-column form) لەر، ئىچىدە Pure Grids ئېلېمېنتى بولغان form (جەدۋەل) نى ئۆز ئىچىگە ئالغان. تۆۋەندىكى بولسا ئارقىمۇ-ئارقا كەلگەن جەۋەل (stacked form) گە Pure Grids ئېلېمېنتى ئىشلىتىشنىڭ مىسالى.


Input رازمېرى (Input Sizing)

Pure Grids غا ئوخشاش ئىچىدە سىنتاكىس (گرامماتىكا) لىق ئاقار جىسىم رازمېرى (ئاقما رازمېر) بار بولغان input ئېلېمېنتىنى كۆرسىتىدۇ. pure-input-* تۈر (class) نامىنى ئىشلىتىپلا بۇ ئېلېمېنتلارنى قۇرالايسىز.

Input رازمېرىنى تېخىمۇ ئىچكىرلىگەن ھالدا grid كونتېينىر(containers) ئىچىگە ئوراپ كونتۇرۇل قىلالايسىز. تۆۋەندىكى مىسالدا، input ئېلېمېنتلىرىنىڭ ئىچىدە pure-input-1 تۈر (class) نامى بار، بىراق ئۇلار ئالاھىدە بىر grid class ئارقىلىق
ئىچىگە ئورالغان .


Required Inputs لىرى

require غا ئوخشاش form كونتۇرولى ئارقىلىق بەلگە قويۇلۇپ، required ئېنىقلىغۇچىسى قوشۇلغان.


پاسسىپ بولغان Input لار

Disabled ئېنىقلىغۇچىسى قوشۇلغان، پاسسىپ بولغان form كونتۇرول (control) نى كۆرسىتىدۇ.


يۇمىلاق (ئايلانما) شەكىللىك Input لار

تۈر (classname) نامى pure-input-rounded غا قوشۇلۇپ، بۇلۇڭلىرى (قىرلىرى) يۇمىلاقلاشتۇرۇلغان form كونتۇرولىنى كۆرسىتىدۇ.

Checkboxes ۋە Radios لار

pure-checkbox ياكى pure-radio تۈر (classname) نامىغا قوشۇلۇپ، قېلىپلاشتۇرۇلغان (نورماللاشتۇرۇلغان) ۋە ئورۇنلاشتۇرۇلغان checkboxes ۋە radio input لىرىنى كۆرسىتىدۇ.
