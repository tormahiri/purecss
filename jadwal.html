Tables
CSS دا ئىشلەنگەن ئاددى بولغان HTML (Tables) جەدۋىلى
Simple CSS for HTML Tables

ئادەتتىكى جەدۋەل (Default Table)

Pure-table تۈر نامى(classname) قوشۇلغان، بىر دانە HTML جەدۋەل (table) ئۇسلۇبى. بۇ تۈر (class) نىڭ Table ئېلېمىېنىتلىرىغا borders ۋە padding قوشۇلغان ، ۋە بېشى (head) تەكىتلەنگەن (كۆپەيتىلگەن) (يەنى th قوشۇلۇپ head ئالاھىدە تەكىتلەنگەن) .

قىرغاق (border) لىق جەدۋەل (Bordered Table)

بارلىق ھۈجەيرىلىرى (cells) گە گورىزونتال ۋە ۋېرتىكال بولغان قىرغاق (borders) لار قوشۇلغان، ئېلېمېنتىغا pure-table-bordered تۈر (classname) قوشۇلغان.


گورىزونتال قىرغاق (border) لىق جەدۋەل (Table with Horizontal Borders)

ئەگەر پەقەتلا گورىزونتال سىزىقلارنى ياختۇرسىڭىز،
ئېلېمېنتىغا pure-table-horizontal تۈر نامى (classname) نى قوشۇڭ.


يوللۇق (يول-يول سىزىقلىق) جەدۋەل (Striped Table)

ئەگەر قۇرلارنى ئاسانلا پەرىق ئەتكىلى بولىدىغان بولسا چوڭ تىپتىكى جەدۋەللەرنى ئوڭايلا قاراپ ئانالىز قىلىغىلى بولىدۇ. ھەربىر ئېلېمېنتىغا pure-table-odd تۈر نامى (classname) نى قوشۇش ئارقىلىق قۇرنىڭ تەگلىك رەڭگىنى ئۆزگەرتىپ ۋە (zebra) تاغىل ئات ئۇسلۇب ئۈنۈمىنى ياراتقىلى بولىدۇ. دىققەت: CSS 3 نى قوللايدىغان تور كۆرگۈدە nth-child يالغان تاللاش بىر ئاددى يول ئىشلىتىلگەن.
ئېلېمېنتىغا pure-table-striped تۈر نامى (classname) قوشقىلى بولىدۇ ۋە تاغىل ئات يول سىزىقچىلىرى ئاپتۇماتىك پەيدا بولىدۇ. بۇ خىل يول IE 8 (Internet Explorer 8) ياكى تۆۋەن بولغان تور كۆرگۈلەردە خىزمەت قىلالمايدۇ.
