جەملەش (Rollups)

Pure دىكى ئۆز ئىچىگە ئالغان بارلىق مودىللارنى جەملەش. بۇ يەردە ئىككى خىل جەملەشنى ئىشلىتىشكە بولىدۇ، بىرى رەتلىك (responsive) لىق جەملەش، يەنە بىرى (responsive) سىز جەملەش. ئۆزىڭىزگە ماس كەلگەن تۈرنى تاللىۋىلىڭ.

مۇستەقىل مودىللار (Individual Modules)

يەنە بۇ مودىللارنى يەككە ھالەتتە بىر تەرەپ قىلالايسىز. Yahoo! CDN گۇرۇپىلار بويىچە بىر تەرەپ قىلىشنى قوللايدۇ، شۇڭلاشقا تەكلىپ- پىكىرلىرىڭىزنى گۇرۇپىلاشتۇرالىغاندىن سىرت يەنە بىر دانە CSS  ھۆججىتىگىمۇ ئېرىشەلەيسىز. بۇ يەردە بىر CSS نىلا ئۆز ئىچىگە ئالغان Base (ئاساس)، Grids (كاتەكچە)، ۋە Forms (ئۇزىتىش جەدۋىلى) گۇرۇپلاشتۇرۇلغان URL مىسالى بار:

تېرە (Skinning)

Pure بىر ئاددىي ئۇسلۇب بىلەن بىرگە كېلىپ شەخسىي CSS دا ئۇنى زاكاز قىلىشقا رۇخسەت قىلىدۇ. ئەگەر بىر تېمىنى تەرەققىي قىلدۇرۇشتا ياردەمگە مۇھتاج بولسىڭىز YUI Skin Builder (YUI تېرە پەيدا قىلغۇچ) ئىشلىتىشكە ئەپلىك بولغان ياخشى قورال بولالايدۇ. بىرنى چېكىش بىلەنلا، Skin Builder (تېرە پەيدا قىلغۇچ) بارلىق زۆرۈر بولغان CSS نىڭ pure تېمىلىرىنى بارلىققا كەلتۈرەلەيدۇ.
