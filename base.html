ئاساس

pure دىكى بارلىق مودىللار normalize.css دە قۇرۇلغان بولۇپ،بىزنىڭ ئاساسى لايھەرىمىز ھەرتۈرلۈك توركۆرگۈچلەردە ئوخشاش ئىجرابولۇشنى بىرتەرەپ قىلىش ئۈچۈندۇر،بىز Normalize v1.x نى ئىشلەتتۇق،بۇ Internet Explorer نىڭ كونا نەشىرلىرنى قوللىغاچقا بىزبۇنى شۇنچىلىك ياخشى كۆرىمىز،شۇڭلاشقا بىز ئۇنى CDN دە ئورۇنلاشتۇردۇق،سىز بىتىڭىزدە link ئېلمىنتىنى قوشۇش ئارقىلىق Normalize.css نى تارتىپ چىقىرالايسىز 

<link rel="stylesheet" href="http://yui.yahooapis.com/pure/0.2.0/base-min.css">

Normalize.css ھەققىدە كىچىككىنە توختايلى

Normalize.css بولساNicolas Gallagher ۋە Jonathan Neal تەرىپىدىن كەلگەن ئوچۇق مەنبەلىك تۈردۇر Normalize.css كچىك css ھۆججىتى بولۇپ ئەسلىدىكى html ئېلمىنلىرىغا ئۇسلۇب بىرىشدە ھەرتۈرلۈك توركۆرگۈچلەردە تەڭ ئىجرابولۇشنى تەمىنلەيدۇ،بۇ ئەنئەنىۋى css نىڭ ئورنىغا دەسستىلىپ html5 گەماسلاشقان Normalize.css ياخشى پايدىلنىش ماتېرياللىرى بىلەن كەلگەن ۋە يىتەكچى ئارقىلىق ئەنئەنىۋى css ئورنىدىن قانداق پەرقى بارلىقىنى چۈشەندۈرۈپ بىرىدۇ،شۇنداقلا سىز YUI نىڭCSSNormalize مودىلىنىمۇ تەكشۈرەلەيسىز،ئۇ كونتېكىست نۇسخىسى بىلەن ئۇنىۋىرسال Normalize نى ئۆزئىچىگە ئالىدۇ، Normalize نىڭ كونتېكىست نۇسخسى پەقەت نورماللاشتۇرۇلغان ئۇسلۇب ئېلمىنتىگە قوللىنىلىدۇ،بۇ yui3-normalized نامىدكى تۈر(class)ئىچدىكى ئېلمىنتىڭ بالىسىدۇر.

خەت ئۇسلۇبى

Pure نىڭ يىنىكلىكى ۋە كىڭەيتىشچانلىقىنى ساقلاپ قىلىش ئۈچۈن بىز Normalize تەرىپدىن تەمىنلەنگەن ئاساسى لايىھەدىن ئوشۇق ھەرقانداق بىرخەت ئۇسلۇبى شەكلىنى قوشمىدۇق،بۇدىگەنلىك سىز ئوخشىمىغان ئۆلچەمدە headings(ماۋزۇ) ۋە نەقىل خەت شەكلى،جەدەۋەل،ۋە جەدۋەل بەلگىلەش قاتارلىق تېخىمۇ كۆپ نەرسىلەرگە ئىرىشەلەيسىز،بىراق ئۇلارنىڭ ئۇسلۇب شەكلى كچىك ۋە قاپلاپ يىزشىقا ئوڭاي

جەدۋەل

جەدۋەل بىرقانچە خىل شەكلىدە كىلىدۇ،بۇيەردە رەتلىك رەتسىز جەدۋەل،رەتلىك جەدۋەل،ۋە ئىنىقلىما جەدۋەللەر بار،Normalize بۇجەدۋەللەرنىڭ ھەرتۈرلۈك توركۆرگۈچلەردە ئوخشاش ئىجرابولۇشنى ۋە ئاساسى ئۇسلۇب بىرىشنى تەمىنلەيدۇ،سىزبۇئۇسلۇبلارنى ئالاھىدە پىروگىراممىڭىزدا ئىشلىتەلەيسىز،ئەگەربۇجەدۋەلنى ئىشلىتىپ تىزىملىك(menu)ياسىماقچى بولسىڭىز

نەقىل خەت شەكلى

نەقىل خەت شەكلى خەتنى ئوراپ تۇرىدۇ،ئۇ ئەسلى ھالىتىدە نەقىل ئىلنغان خەت شەكلىگە ۋەكىللىق قىلىدۇ,ئەسلىدە بۇ نەقىل خەت شەكلى دائىم خەت شەكلىگە ئوخشاپ كېتىدۇ،بىراق بۇنەقىل شەكلى قىسىلغان،بۇسىزگە خەت ئۇسلۇبىنى بەلگىلەش ئەركىنلىكىنى بىرىدۇ،تۆۋەندىكىسى بولسا بۇلارنىڭ بىرسى

ئىنسانغا نىسبەتەن چىرايلىق بىرقەدەم،ئىنسانىيەتكە نىسبەتەن چوڭ بىرقەدەمدۇر.


ئەسلى شەكىلنىڭ يىتەرسىزلىكىدىن پايدىلنىپ بىزبۇنى بىكتىڭزدە ئازراق ئوخشىمايدىغان نەقىل خەت شەكلى كۆرۈنىشىى قىلىپ بەلگىلىدۇق،مەقسىتىمىز بىكىتڭىزنىڭ كۆرۈنۈشىگە ئاساسەن مۇشۇنداق قىلىدۇرۇش

ئادرېس

ئەسلى ھالىتىدە ئالاقە ئۇچۇرلىرى ئۈچۈن قوللىنىلىدۇ،ئادرېس نىڭ ئالاھىدە ئۇسلۇبى يوق،مەيلى قانداق بولۇسۇن،بۇلار Normalize ئارقىلىق ئىچكى بىرلەشمە ئۇسلۇب مىقدارىنى ئىشلىتىپ بىزىلىدۇ.

ئىچكى بىرلەشمە شەكلى

بۇيەردە نورغۇنلىغان ئىشلەتكىلى بولدىغان ئىچكى بىرلەشمە شەكىل بار،بۇيەردىكى Normalize ئارخىپىغا قاراپ چىقىڭ

Normalize دىن ھالقىش

Normalize بىرەرتۈرىڭىز ئۈچۈن ئسىل باشلاش نۇقىتىسى بولالايدۇ،بىراق html ئېلمىنتلىرى شۇنىڭدەك ئۇچۇر جەدۋىلى،جەدۋەل،تىزىملىك يىتەكچىسى Normalize تەرىپىدىن تەمىنلەنگەن ئىقتىداردىنمۇ بەكرەك پاسۇنلاشنى تەلەپ قىلىدۇ. بىزبۇنى تېخىمۇ كۆركەم ھىس تۇيغۇلۇق قىلىپ چىقىش بىلەن بىرگە كىچىكلىك ھالىتىنى تولۇق ساقلاپ قالدۇق،شۇڭلاشقا سىزبىكتىڭىز ياكى كىرەكلىك ئەپلىرىڭىز ئۈچۈن ئۆزىڭىز بەلگىلىۋالسىڭىز بولىدۇ. ئۇچۇر جەدۋىلى،جەدۋەل،ۋە تىزىملىك يىتەكچىسى ئۈچۈن بۇيەرنى تەكشۈرۈپ كۆرۈڭ
