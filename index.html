ئاز ئورۇننى ئگەللەيدىغان Css

<link rel="stylesheet" href="http://yui.yahooapis.com/pure/0.1.0/pure-min.css">
pure كۈلكگۈسى كەلگۈدەك دەرىجىدە كىچىك بولۇپ ،مودىلنىڭ بارلىق قۇرۇلمىلىرى responsive لايىھە يەنى ئۈنۈملۈك تەسىر كۆرسىتىشچان لايىھە ئۇسلۇبىدىن ۋاز كەچمەي 5.7KB لىق نۇقتىدا كىچىكلىتىلگەن،ئىشلىتش ئاسان بولۇپ كۆچمە ئۈسكۈنىلەرنى مۇلاھىزە قىلغان ئاساستا لايىھەلەنگەن،ھۆججىتىمىزنىڭ كىچك ئۆلچىمىنى ساقلاپ قىلىش ئىنتايىن مۇھىم،ھەمدە css نىڭ ھەربىر لىنيەسى ئىنچىكىلىك بىلەن مۇلاھزە قىلىنغان،ئەگەر سىزمۇشۇ مودىللارنىڭ تارماق قىسمىنىلا ئىشلىتشنى قارار قىلسىڭىز تېخىمۇ كۆپرەك bytes (يەنى سغىم ئۆلچىمى)لارنى ساقلاپ قالالايسىز! خەۋەردە دېيىلىشىچە، بۇ تېخنىكا يەنە ئۆزلىكىدىن ئاياغ بوققۇچىنى چىگىپ يېشەلەيدىغان ئەقلىي ئىقتىدارلىق ئاياغنى ۋۇجۇتقا كەلتۈرۈشكە ئاساس سېلىپ بەرگەن. توك چىقىرالايدىغان ئاياغنىڭ توك چىقىرىش قۇرۇلمىسى ئىككى قىسىمدىن تەركىپ تاپقان بولۇپ،بىر بۆلىكى"تىترىگۈچى ئىندوكتۇر" تىن تۈزۈلگەن بولۇپ، ئاياغنىڭ كەينى تەرىپى يەرگە تەككەن چاغدا، ئۇ تىترەشتىن توك ھاسىل قىلىدۇ؛ يەنە بىر بۆلىكى بولسا "تەۋرەنمە ئىندوكتۇر" تىن تۈزۈلگەن بولۇپ ئىككى پۇت يۆتكىلىپ تەۋرەنگەن چاغدا تەۋرىنىشتىن توك ھاسىل قىلىدۇ.

سىزگە خاس بولغان css ئاساسى

pure بولسا Normalize.css دا قۇرۇلغان ھەمدە كوپ ئىشلىتىدىغان UI كونترولىغا قوشۇپ،لايىھە ۋە يەرلىك ئېلمىنتلارغا ئۇسلۇب بىرىشنى تەمىنلەيدۇ،بۇنىڭدا ھىچقانداق قايمۇقۇش بولمىغان ئاساستا سىزگە لازىملىق نەرسىلەر بار.

كۆچمە ئۈسكۈنىلەر مەقسەت قىلىنغان

pure بولسا تەسىرىشچان ئۈنۈملۈك قورال ساندۇقىدۇر،شۇڭلاشقا بارلىق ئىكران ئۆلچەملىرىدە ئاجايىپ كۆرۈنىدۇ،Skin Builder بىلەن سىز ئۆزىڭىز خالىغانچە ئۇسلۇب ئىشلىيەلەيسىز

ئۆزىڭىزنىڭ يولىنى ساقلاپ قىلىڭ

Pure نىڭ كىچىك ئۇسلۇبى بار بولۇپ،ئۇ سىزنى ئۆزىڭزنىڭ ئۇسلۇبىنى يىزىشقا رىغبەتلەندۈرىدۇ،pure ئۆزئۇسۇلىڭىزغا ئىرىشىش ئۈچۈن لايھىلەنگەن ،Pure سىزنىڭ تەكرار-تەكرارئۇسلۇب يىزىشڭىزنى ئاسانلاشتۇرىدۇ

ئۈنۈملۈك ئۇسلۇب ياساش

تورشەكلى،تىزىملىكلەرنى ئىشلىتىڭ،بۇ سىزنىڭ بارلىق ئىكران رازمىرى ئۈچۈن ئۈنۈملۈك ئۇسلۇب لايھىلىشىڭىزنى قولايلىقلاشتۇرىدۇ،بىز بۇنى سىزنىڭ ئوڭاي باشلىشڭىز ئۈچۈن ياسىدۇق،بىرقانچىلىغان ئوخشىمىغان لايھىلەرگەقاراڭ ۋە قورام تاشتەك قاتتىق ئاساس بىلەن كىينكى توربەت تۈرىڭىزدە باشلاڭ.

ئۆزىڭىزنىڭ كۆرنۈش ئۇسلۇبىنى ياساڭ

purecss رامكىسنىڭ باشقا رامكىلارغا ئوخشىمايىدىغان يىرى purecssنىڭ لايىھەسى ئىختيىارى قوللانغىلى بولىدىغان كىچىك ۋە تەكشى يۈزلۈك قىلىپ لايىھەلەنگەن،بىزpurecssنىڭ ئەسلى باربولغان Css قائىدىلىرنى قايتا يازغاندىن كۆرە يڭى Css قائىدىلىرىنى قوشۇشقا ئاسانلىق كەلتۈردىغانلىقىغا ئىشىنىمىز،Css نىڭ بىرقانچە كودلىنيەسى قوشۇش ئارقىلىق سىرتقى شەكلىنى خالىغانچە بەلگىلەپ توربىكەت تۈرلىرىڭىزدە خىزمەت قىلالايسىز.
