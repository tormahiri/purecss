تىزىملىك يىتەكچىسى

تەكشى تىزىملىك يىتەكچسى ياساش ئۈچۈن pure-menu-horizontal تۈر ئسىمىنى قوشۇڭ،سىز pure-menu-selected تۈرنامىنى list ئېلمىنتىغا قوشۇش ئارقىلىق نۆۋەتتە ئاكتىپ ھەرىكەتتە بولغان List ئېلمىنتىغا بەلگە ئۇرالايسىز.

تىك يۆلىنشتىكى تىزىملىك يىتەكچىسى

تىزىملىكلەر ئەسلىدە تىك يۆلىنىشدە كىلىدۇ،سىز پەقەت pure-menu تۈرنامىنى ئۇسلۇبلايدىغان تىزىملىك يىتەكچىسىگە قوشۇشىڭىز كىرەك، ھەمدە pure-menu-open تۈرنامىنىڭ كۆرنۈپ تۇرىىشغا كاپالەتلىك قىلىشڭىز كىرەك


ئاسما تىزىملىك يىتەكچىسى

ئاسما تىزىملىك قوشۇش Javascript نىڭ ئىشلىتىلىشنى تەلەپ قىلىدۇ،مەسىلەن Yahoo Yui Javascript ئامبىرىنىڭ y.menu مودىلى ئاسما تىزىملىك ئىقتىدارىنى قوشىدۇ ،شۇنداقلا Javascript لىق تىزىملىك يىتەكچىسى ھاسىللاش ئىقتىدارىغا ئىگە


چەكلەنگەن تۈرھالىتدىكى تىزىملىك يىتەكچىسى

بىرئۇلانمىنى چەكلەنگەن ھالەتتە چىقىرىش ئۈچۈن ئىلمتىنى pure-menu-disabledتۈرنامى بىلەن قوشۇڭ،بۇۋاقىتتا چەكلەنگەن تۈر پەيدا غۇۋا شەكىلدە پەيدا بولىدۇ دە شۇنداقلا Hover شەكلىگە ۋارىسلىق قىلمايدۇ.


بەت ئايرىغۇچ

بەت ئايرىغۇچ كونترولى ياساش ئۈچۈن pure-paginator تۈرنامىنى
    ئېلمىنتىگە قوشۇڭ،بەت ئايرىغۇچنىڭ كونۇپكا شەكلىدە كۆرنۈشى ئۈچۈن pure-button تۈرنامىنى قوشۇڭ.ئادەتتكىچە بەت ئايرىغۇچ ئۈچۈن بۇ تۈرنامىنى ئۆچۈرۋەتسىڭىز بولىدۇ.
